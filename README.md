# Mercury Visions - Premium Liquid Chrome Hero Section

A highly realistic, real-time liquid mercury hero background built with **Vue 3**, **Three.js**, and custom **GLSL Raymarching Shaders**. 

https://hero-section-3.vercel.app/

## ‚ú® Features
- **High-Fidelity Mercury Engine**: Procedural raymarching with 25+ dynamic metaballs.
- **Organic Liquid Physics**: Features "Squash & Stretch" morphing to simulate metallic surface tension.
- **Studio-Grade Shading**: Multi-layer specular highlights (sharp glints + soft bloom) and terminal-contrast studio reflections.
- **Precision Typography**: Minimalist, perfectly centered UI layout for a premium "Spatial Computing" aesthetic.
- **Interactive Field**: The mercury surface and reflection highlights attract and bend towards the mouse cursor.

### Prerequisites
- Node.js (v16+)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Vann4799/Hero-Section-3.git
   cd Hero-Section-3
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
